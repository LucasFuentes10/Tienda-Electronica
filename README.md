"EcoTech – Aplicación de Escritorio en Java"

Tienda Electrónica es un proyecto desarrollado completamente en Java, que implementa un sistema de gestión para una tienda de productos electrónicos. 
Incluye administración de usuarios, manejo de productos, búsquedas por categoría y un módulo básico de carrito de compras.

El sistema no utiliza una base de datos, sino que almacena la información en archivos .ini, lo que permite persistencia ligera sin dependencias externas.

"Características técnicas principales"
  
  Gestión de Usuarios
  
  Alta, baja y modificación de usuarios.
  Persistencia en archivos usuarios.ini.
Lectura y escritura mediante estructuras simples basadas en clave–valor.
  
Gestión de Productos
  
Carga, edición y eliminación de productos.
Clasificación por categorías para facilitar la búsqueda.
Persistencia en productos.ini con formato similar al de los usuarios.
  
  Carrito de Compras
  
  Selección de productos.
  Agregado/eliminación de ítems del carrito.
  Simulación de compra final.
